## Descripción del proyecto

Sistema web desarrollado para el emprendimiento **Peques**, enfocado en la gestión integral de inventario y la comercialización de productos a través de una plataforma digital.

El sistema está diseñado para optimizar procesos administrativos y mejorar la experiencia de compra mediante una interfaz moderna, eficiente y adaptable a diferentes dispositivos.

El sistema incluye:

* Panel administrativo para la gestión de productos, usuarios y pedidos
* Plataforma de tienda en línea para clientes
* Módulo de gestión de inventario
* Flujo completo de compra dentro de la plataforma

## Tecnologías utilizadas

* React
* Vite
* Tailwind CSS
* React Router DOM
* Lucide React
* Sonner
* React Hook Form
* Framer Motion

## Instalación

Clonar el repositorio:
git clone https://github.com/TU-USUARIO/SGITP_PTC2026.git

Instalar dependencias:
npm install

Ejecutar el proyecto:
npm run dev

## Configuración del entorno

* Proyecto inicializado con Vite para un entorno de desarrollo rápido y optimizado
* Tailwind CSS integrado mediante @tailwindcss/vite
* Arquitectura basada en componentes reutilizables
* Manejo de rutas con React Router DOM
* Gestión de formularios con React Hook Form
* Sistema de notificaciones con Sonner
